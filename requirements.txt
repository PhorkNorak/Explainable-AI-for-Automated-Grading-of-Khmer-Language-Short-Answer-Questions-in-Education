# Recommended install on HPC (Linux, CUDA GPU):
#
#   1. Check your NVIDIA driver / supported CUDA version:
#        nvidia-smi   # see "CUDA Version" in the top-right corner
#
#   2. Install torch with the matching CUDA wheel BEFORE the rest:
#        pip install --upgrade pip
#        # driver 12.4+:  use cu124 (most compatible)
#        pip install torch --index-url https://download.pytorch.org/whl/cu124
#        # driver 12.8:   use cu128 (latest 12.x)
#        # driver 12.1:   use cu121
#        # driver 11.8:   use cu118
#
#   3. Install everything else:
#        pip install -r requirements.txt
#
# ⚠ Do NOT run `pip install -r requirements.txt` directly without step 2 —
#   PyPI's default torch wheel uses the LATEST CUDA (currently cu130) which
#   requires a fresh GPU driver. If your driver is older you'll get the
#   "NVIDIA driver too old" warning and CUDA will be unavailable.
#
# On Windows the default wheel is CPU-only; use the cu124 index to force CUDA.
